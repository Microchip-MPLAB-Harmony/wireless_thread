Introduction

The Standalone Open Thread Application example demonstrates bidirectional communication using UDP API’s in open thread. The application example consists of FTD’s and MTD’s.

The application supports the following device types.

  1. 1.FTD as router
  2. 2.FTD as leader
  3. 3.MTD as MED
  4. 4.MTD as SED
